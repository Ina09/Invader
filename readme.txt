■自機

■ブロック
HP=一文字につき4
描画(壊れていなければそのまま)
弾の当たり判定(当たっていればそのブロックのHPを呼び出した相手の攻撃力分減らす)

■敵
HP=一文字につき1
ポイント(段ごとに設定)

移動(移動したときに段がブロックに接触していないか確認する必要がある)
移動速度計算(敵の数で判断(段が変わるときに速度を計算しなおす))
当たり判定(点数を返す)

	移動処理の時にブロックのオブジェクトをとってきて下の列から当たり判定をする
